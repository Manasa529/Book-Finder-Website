# 📚 Book Finder Website

Book Finder is a simple web application that allows users to search for books and view book details such as title, author, and cover image.

This project is built using **HTML, CSS, and JavaScript** and uses a public books API to fetch book data dynamically.

---

## 🚀 Features

- Search for books by title
- Display book cover images
- Show book author information
- Responsive and simple user interface
- Fetch book data using JavaScript API requests

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Books API (Google Books API)
